# Slip

[![Build Status](https://travis-ci.org/dbrabera/slip.svg?branch=master)](https://travis-ci.org/dbrabera/slip)

The Slip programming language

## Usage

```
$ slip
slip> (println "It's ALIVE!")
It's ALIVE!
$ cat hello.sp
(println "It's ALIVE!")
$ slip hellp.sp
It's ALIVE!
```
